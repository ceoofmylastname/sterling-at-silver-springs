## Glass-Morphism Integration Note

When using these images on the website, apply a subtle glass-morphic frosted border treatment to bridge the gap between photorealistic imagery and the site's modern UI:

```css
.hero-image-treatment {
  border-radius: 16px;
  border: 1px solid rgba(255, 255, 255, 0.2);
  box-shadow:
    0 8px 32px rgba(0, 0, 0, 0.1),
    inset 0 0 0 1px rgba(255, 255, 255, 0.1);
  overflow: hidden;
}
```

For hero backgrounds, overlay a gradient that includes the site's primary blue at reduced opacity to maintain brand cohesion:

```css
.hero-overlay {
  background: linear-gradient(
    160deg,
    rgba(44, 68, 96, 0.7) 0%,
    rgba(61, 90, 128, 0.45) 35%,
    rgba(43, 45, 66, 0.65) 100%
  );
}
```
