Usage Notes

Embed in Google Sites: Screenshot individual icons or export as PNG from browser dev tools. Each icon renders at retina quality.

Integrate in HTML: Copy any <div class="icon-card"> block directly into the website preview or production pages. The glassmorphism requires backdrop-filter support (Chrome, Safari, Firefox 103+).

Color customization: Change the .ib-* gradient class on the .icon-bubble to swap icon background colors. Nine color themes included: blue, gold, green, red, teal, purple, orange, slate, rose.

Dark mode: The dark preview strip above demonstrates how icons look on dark backgrounds. For full dark-mode support, override --glass-bg and --clay-shadow values as shown in the .dark-strip CSS.
