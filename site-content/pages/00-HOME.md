# HOME PAGE — Sterling at Silver Springs
> Layout instructions for Google Sites. Copy text into the site editor, section by section.

---

## Section 1: Emergency Alert Banner
> **Google Sites instruction:** Add a full-width text box at the very top. Set background to red (#E63946), text to white, bold. When there's no emergency, delete the text inside but keep the empty section hidden or remove it entirely.

**Sample alert text (use when needed):**
```
COMMUNITY ALERT: [Water main repair on Silver Springs Blvd — expect reduced pressure 3/31–4/2. Contact your water provider with questions.]
```

## Section 4: Latest Announcement
> **Google Sites instruction:** Add a section with a heading + short text + button. Update this manually whenever a new announcement is posted.

**Sample content:**
```
[March 2026] — Spring Landscaping Schedule Released
The board has approved the 2026 spring landscaping plan. New plantings begin April 15.
View the full announcement for details on affected areas and temporary path closures.
```

**Button:** Read All Announcements → [link to Announcements page]
