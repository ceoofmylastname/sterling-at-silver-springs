## Section 4: Topic Guide
> **Google Sites instruction:** Help residents choose the right topic. Collapsible section or simple list.

**Not sure which topic to choose? Here's a quick guide:**

| If your question is about... | Select this topic: |
|-----------------------------|--------------------|
| Board correspondence, documents, account changes, general board questions | **Administrative** |
| Community amenities (playground, common areas, fitness equipment) | **Amenities** |
| Exterior modifications (paint, additions, solar panels, satellite dishes) | **Architectural** |
| Clubhouse reservations, access, or issues | **Clubhouse** |
| Community fences — damage, repairs, replacements | **Fencing** |
| Gate access, security cameras, security concerns | **Gates/Security** |
| Gate keys, fobs, remotes — lost, broken, or new | **Key/Fob** |
| Trees, shrubs, grass, common area plantings | **Landscaping** |
| Streetlights, pathway lights, common area lighting | **Lighting** |
| Mailbox damage, locks, cluster box issues | **Mailboxes** |
| General repairs — anything not covered by other categories | **Maintenance** |
| Roads, sidewalks, curbs, driveways in common areas | **Pavement** |
| Pool or spa — access, cleanliness, equipment, hours | **Pool/Spa** |
| Community signs — damaged, missing, or new requests | **Signs** |
| Trash bins, pickup issues, bulk waste, recycling | **Trash** |
| Reporting a community rule violation | **Violations** |
| Sprinklers, irrigation leaks, water features, drainage | **Water/Irrigation** |
